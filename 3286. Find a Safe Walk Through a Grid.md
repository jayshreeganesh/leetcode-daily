# Multi-Language 0-1 BFS Portfolio

## Core Algorithmic Framework

The problem requires finding a path from $(0,0)$ to $(m-1, n-1)$ while minimizing the total cost (health lost). Because grid edge weights are strictly binary ($0$ if a cell is empty, $1$ if it contains an obstacle), a standard Dijkstra algorithm with an $\mathcal{O}(E \log V)$ priority queue introduces unnecessary logarithmic overhead.

Instead, we use a **0-1 Breadth-First Search (0-1 BFS)** framework using a Double-Ended Queue (Deque):

* **Weight = 0:** Prepend to the front of the queue (`push_front`). This node is processed with the highest priority since it does not increase our cost accumulation.
* **Weight = 1:** Append to the back of the queue (`push_back`).

For languages lacking built-in Deque structures, a fast, flat vector with a centered index pointer provides true $\mathcal{O}(1)$ insertion boundaries across both ends without triggering sequential element shifting.

## 1. C++23

Uses `std::deque` for dynamic push/pop operations at both ends, alongside modern structural auto-unwrapping.

```cpp
#include <vector>
#include <deque>

class Solution {
public:
    bool findSafeWalk(std::vector<std::vector<int>>& grid, int health) {
        int m = grid.size();
        int n = grid[0].size();
        
        // 2D distance tracking initialized to an arbitrary large sentinel value (Infinity)
        std::vector<std::vector<int>> dist(m, std::vector<int>(n, 1e9));
        std::deque<std::pair<int, int>> dq;
        
        // Initialize origin
        dist[0][0] = grid[0][0];
        dq.push_back({0, 0});
        
        // Cardinal direction step mapping
        int dRow[] = {-1, 1, 0, 0};
        int dCol[] = {0, 0, -1, 1};
        
        while (!dq.empty()) {
            auto [r, c] = dq.front();
            dq.pop_front();
            
            if (r == m - 1 && c == n - 1) break; // Optimization target met
            
            for (int i = 0; i < 4; ++i) {
                int nr = r + dRow[i];
                int nc = c + dCol[i];
                
                if (nr >= 0 && nr < m && nc >= 0 && nc < n) {
                    int weight = grid[nr][nc];
                    if (dist[r][c] + weight < dist[nr][nc]) {
                        dist[nr][nc] = dist[r][c] + weight;
                        
                        // 0-1 BFS placement logic
                        if (weight == 0) {
                            dq.push_front({nr, nc});
                        } else {
                            dq.push_back({nr, nc});
                        }
                    }
                }
            }
        }
        return health - dist[m - 1][n - 1] >= 1;
    }
};

```

### Line-by-Line Explanation

* **Lines 9-10:** Extract the matrix dimensions `m` and `n`.
* **Line 13:** Pre-allocate `dist` tracking vector layer populated with a $10^9$ sentinel.
* **Lines 17-18:** Queue up the starting block $(0,0)$ and register its initial value.
* **Line 24:** Deconstruct pair references using structured binding syntax.
* **Lines 35-39:** If the weight is `0`, prepend to the head using `push_front` to evaluate it next; otherwise, append to the tail using `push_back`.
* **Line 46:** Verify that the remaining health reserve stays at $\ge 1$.

## 3. Python 3

Leverages `collections.deque` written natively in optimized C code under the hood.

```python
from collections import deque
from typing import List

class Solution:
    def findSafeWalk(self, grid: List[List[int]], health: int) -> bool:
        m, n = len(grid), len(grid[0])
        dist = [[float('inf')] * n for _ in range(m)]
        
        dq = deque()
        dist[0][0] = grid[0][0]
        dq.append((0, 0))
        
        while dq:
            r, c = dq.popleft()
            
            if r == m - 1 and c == n - 1:
                break
                
            for dr, dc in [(-1, 0), (1, 0), (0, -1), (0, 1)]:
                nr, nc = r + dr, c + dc
                
                if 0 <= nr < m and 0 <= nc < n:
                    weight = grid[nr][nc]
                    if dist[r][c] + weight < dist[nr][nc]:
                        dist[nr][nc] = dist[r][c] + weight
                        if weight == 0:
                            dq.appendleft((nr, nc))
                        else:
                            dq.append((nr, nc))
                            
        return health - dist[m - 1][n - 1] >= 1

```

### Line-by-Line Explanation

* **Line 7:** Use list comprehension to allocate independent row tracks for `dist`.
* **Line 13:** Evict elements from the head via `popleft()`.
* **Line 18:** Linearly loop through bound offset structures natively unpacked inside the `for` statement.
* **Lines 25-28:** Route the cell to the front with `appendleft()` or the back with `append()`.

## 5. JavaScript Node.js 22.14.0

Simulates a double-ended queue using a pre-allocated array with an internal pointer tracking configuration. This avoids the $\mathcal{O}(N)$ reindexing penalty of `Array.prototype.shift()`.

```javascript
var findSafeWalk = function(grid, health) {
    const m = grid.length;
    const n = grid[0].length;
    
    const dist = Array.from({ length: m }, () => Array(n).fill(1e9));
    
    const maxSize = m * n * 4;
    const dq = new Array(maxSize);

    let tail = head;
    
    dist[0][0] = grid[0][0];

    const dCol = [0, 0, -1, 1];
```

```c
                        dqC[head] = nc;

                        dqC[tail] = nc;
```

```elixir
    m = length(grid)
    n = length(hd(grid))

    grid_tuple = grid |> Enum.map(&List.to_tuple/1) |> List.to_tuple()
    dist = for r <- 0..(m - 1), c <- 0..(n - 1), into: %{}, do: {{r, c}, 1_000_000_000}

  defp bfs(dq, head, tail, dist, grid, m, n) do

    else

              else

                t = t + 1
                {acc_dq, acc_dist, h, t}
              end
            else
              {acc_dq, acc_dist, h, t}
            end
          else
            {acc_dq, acc_dist, h, t}
```

```erlang
    Dist;

    {Curr, Dq1} = maps:take(Head, Dq),
    NewHead = Head + 1,
    {R, C} = Curr,

        R =:= M - 1, C =:= N - 1 ->
            Dist;

                                            NewDq = maps:put(NewH, {NR, NC}, AccDq),

                                            NewDq = maps:put(T, {NR, NC}, AccDq),

                                true ->
                                    {AccDq, AccDist, H, T}
                            end;
                        true ->
                            {AccDq, AccDist, H, T}

                {Dq1, Dist, NewHead, Tail},
```
