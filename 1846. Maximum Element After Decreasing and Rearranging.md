# 1846. Maximum Element After Decreasing and Rearranging - All 19 Languages

### Algorithm Explanation
For all languages, the optimal greedy strategy remains exactly the same:
1. **Sort** the array in non-decreasing order.
2. **Initialize** the first element to `1` (to satisfy the first constraint).
3. **Iterate** from the second element to the end. If the current element is greater than the previous element plus one, clamp it to `previous + 1`.
4. **Return** the last element, which will be the maximum possible value.

### 1. C++
```cpp
class Solution {
public:
    int maximumElementAfterDecrementingAndRearranging(vector<int>& arr) {
        sort(arr.begin(), arr.end());
        arr[0] = 1;
        for (int i = 1; i < arr.size(); ++i) {
            if (arr[i] > arr[i - 1] + 1) {
                arr[i] = arr[i - 1] + 1;
            }
        }
        return arr.back();
    }
};
```
