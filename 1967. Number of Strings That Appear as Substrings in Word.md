## 1. C++

### Imperative Solution

```cpp
class Solution {
public:
    int numOfStrings(vector<string>& patterns, string word) {
        int count = 0;
        for (const string& pattern : patterns) {
            // string::npos is returned if the substring is not found
            if (word.find(pattern) != string::npos) {
                count++;
            }
        }
        return count;
    }
};

```

#### Line-by-Line Explanation

* `int count = 0;`: Initializes an integer tracker to record the total number of matched patterns.
* `for (const string& pattern : patterns)`: Iterates through the list of pattern strings by constant reference to avoid unnecessary memory allocations.
* `if (word.find(pattern) != string::npos)`: Uses the standard library `std::string::find` method. If the pattern exists inside `word`, it returns the first match index. If it does not exist, it evaluates to `string::npos`.
* `count++;`: Increments the accumulator if a valid match location is identified.

### Functional Alternative

```cpp
class Solution {
public:
    int numOfStrings(vector<string>& patterns, string word) {
        return count_if(patterns.begin(), patterns.end(), [&](const string& p) {
            return word.find(p) != string::npos;
        });
    }
};

```

### Functional Alternative (Streams API)

